tucson-trailguide
=================

You had best:
- `gem install sass`
- `brew install couchdb` (or something along those lines)

Then you can:

1. Clone the repository: 

        git clone https://github.com/rclark/tucson-trailguide.git

2. Install:

        npm install

3. Run:

        npm start
